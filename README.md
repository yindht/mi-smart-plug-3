# mi-smart-plug-3
Mi Smart Plug 3(WiFi) install tasmota firmware.  

Hardware info:  
CPU:ESP32C3  
Module:MHCWB6S-B
https://manuals.plus/zh-CN/xiaomi/mhcwb6s-b-2-4ghz-wifi-bluetooth-dual-mode-module-manual  


PowerChip:KP15052（5V）  ASM117(3.3V)  
ADC:BL0942  
Relay:hf32fv-16/5hctf590  

GPIO  
IO10: relay  
IO5: LED1  
IO18:LED2  
IO19:Button  
IO6: ADC Serial  
IO7: ADC Serial
